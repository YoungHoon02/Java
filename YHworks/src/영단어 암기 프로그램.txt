단어 관리 [20] (기능 구현이 안되면 코드 내에 변수 등록 함) <JFrame> [세이브/로드는 후순위로]
▪ 암기하고 싶은 단어를 관리함 (파일 시스템 사용) (Save/Load/get/set) (C R U D) [20]
▪ 영어 단어, 한글 뜻, 등록일, 확인일을 기록함

단어 확인 [20]
▪ 단어 하나를 보여주고, 답을 맞추는 클래스or메소드
▪ 한글 뜻이 여러 개인 경우 그 중 하나만 맞추어도 답으로 인정
▪ 답을 맞추는 경우 영어 단어의 확인일을 오늘로 갱신
▪ 답이 틀리는 경우 확인일을 갱신하지 않음

낱말 카드 생성 [20] <JFrame>
▪ 1개의 영어 단어를 확인하는 GUI
▪ 입력 받은 영어 단어를 출력하고, 단어가 맞는지 확인함
▪ 알림창으로 맞고 틀림을 알려줌
▪ 창을 닫아도 메인 창은 종료되면 안됨(dispose?)

영어 단어 암기 Test [20] <JFrame>
▪ 여러 개의 영어 단어를 확인하는 GUI
▪ 입력 받은 영어 단어의 숫자를 확인하고, 사용자에게 시험지를 만들어 줌
▪ 입력 받은 영어 단어의 순서를 랜덤하게 배치함
▪ 제출 시 결과를 보며 주는 창/컴포넌트를 배치

메인 기능 [10] + 객체화 [10]
▪ 현재 등록된 단어의 수를 보여줌 [2]
▪ 단어를 등록 할 수 있는 GUI를 갖고 있음 [2]
▪ 낱말 카드 생성 버튼 [2]
▪ 버튼 클릭 시 낱말 카드 생성 GUI 실행

▪ 영어 단어 암기 Test 버튼 [2]
▪ 평가할 단어의 개수를 사용자로부터 입력 받음 (기본 값 10)
▪ 버튼 클릭 시 영어 단어 암기 GUI 실행
▪ 등록 된 단어 중 확인일이 가장 오래된 단어를 개수만큼 선택
▪ 평과 결과를 리턴 받음
▪ 최근 영어 단어 암기 결과 표현 [2]
